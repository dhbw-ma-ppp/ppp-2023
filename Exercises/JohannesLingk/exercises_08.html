Filling NaN values

By default the age column of the dataset has 177 NaN values. Although we could ignore those values altogether I think it's nicer to fill them with something, for example the mean value of all ages.

In[]:
print(f"Age Column has {titanic_data['Age'].isna().sum()} NaN values")
titanic_data['Age'] = titanic_data['Age'].fillna(titanic_data['Age'].mean())
